# 🌿 EcoData AI - Inteligência Energética

## 📌 Sobre o Projeto
Este repositório contém a segunda etapa do **CheckPoint 1** da disciplina de *Application Development* da FIAP [cite: 2026-03-12]. A **EcoData AI** é uma startup fictícia concebida para oferecer soluções de otimização de consumo elétrico utilizando Inteligência Artificial Generativa e análise de dados [cite: 2026-01-02].

O projeto demonstra a aplicação prática de **HTML5 semântico** e **CSS3 avançado**, com um sistema de navegação *Single Page Application* (SPA) baseado em seletores `:target`, eliminando a necessidade de JavaScript para a troca de abas [cite: 2026-03-12].

---

## 🚀 Conceitos de Startup (Modelo de Negócio)
A EcoData AI opera na interseção entre sustentabilidade e tecnologia de ponta [cite: 2026-01-02]:
- **Monitoramento Preditivo**: Uso de **Machine Learning** para antecipar picos de consumo [cite: 2026-02-12].
- **Interface de Voz**: Integração com APIs da **OpenAI** para transcrição e síntese de áudio (STT/TTS) [cite: 2025-10-23].
- **Agentes Inteligentes**: Automação de notificações e relatórios via **Telegram API** [cite: 2025-10-23].

---

## 🛠️ Detalhes Técnicos do Front-end
- **Layout Moderno**: Uso de **CSS Grid** e **Flexbox** para estruturação de cards e tabelas [cite: 2026-03-12].
- **Estilização**: Paleta de cores baseada em tons de verde e cinza corporativo, focada na identidade visual de empresas de energia [cite: 2026-03-12].
- **Organização Semântica**: Implementação de tags estruturais (`<header>`, `<main>`, `<section>`, `<footer>`) para melhor SEO e acessibilidade [cite: 2026-03-12].
